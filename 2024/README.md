## 컨트리뷰션 주제

1. 단위 테스트
   - DAO 단위 테스트
   - ServiceImpl 단위 테스트
   - Controller 단위 테스트
   - 셀레늄 단위 테스트
2. 통합 테스트
   - Controller 통합 테스트
   - 셀레늄 통합 테스트
3. 시큐어코딩
   - 시큐어코딩 Exception 제거
   - 시큐어코딩 일련번호 PK 파라미터 암복호화
   - 시큐어코딩 PMD 진단/조치
4. 검색조건 유지
5. 롬복 생성자 기반 종속성 주입
