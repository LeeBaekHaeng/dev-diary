PR 내용
```

이유
- Spring Framework 5.2 이상에서는 @Bean 메서드가 반드시 public일 필요가 없습니다.
- 패키지 접근자(package-private)를 사용하면 불필요한 API 노출을 줄일 수 있습니다.
- 캡슐화 향상 및 정적 분석 규칙 준수.
- 기능 변경 없는 코드 개선

```
